• There are two main methods of submitting the form information

    • GET

    • POST

  • GET requests a representation of the specified resource.

  • With our simple HTML forms, GET will send back the inforamtion to our action URL.

  • POST submits data to be processed.

  • We'll be using POST once we learn how to connect a database with Flask to the form.

  • For now, we'll supply an action url for a redirect after submitting the form.
