CSS Classes and Ids

  • We know that we can select every element of a certain tag type in our CSS file
    • Every p element
    • Every h1 element

  • But what do we do if we only want to select some of these? Or just one?

  • We've explored "divs" and "spans but these options limit our structure in HTML, having to make sure everything is in the same "div"

  • The solution to these issues is the use of "classes" and "ids" in our CSS.

  • Classes are used to assign a CSS style to multiple differetn elements throughout the HTML.

  • Ids are used to apply a style to a single element

  • Ids should be used fro unique elements
